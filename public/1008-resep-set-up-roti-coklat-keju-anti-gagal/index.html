Bahan-bahan dan bumbu yang dibutuhkan untuk pembuatan Set up roti coklat keju:
  1. Siapkan 10 lembar roti tawar buang pinggirnya
  2. Ambil 500 liter susu full cream
  3. Siapkan 3 sdm maizena
  4. Gunakan 200 grm gula pasir
  5. Sediakan 2 sdm coklat cair
  6. Ambil secukupnya keju tuk taburan

Langkah-langkah membuat Set up roti coklat keju:
  1. Masukan susu, gula, maizena dan coklat ke dlm panci aduk rata
  2. Masak di atas api sedang sambil terus di aduk sampe matang
  3. Masukan roti yg sudah di potong potong ke dlm wadah tata lalu siram dgn saus susu tata lagi roti masukan saus coklat terus sampe habis
  4. Taburi keju di atas nya
  5. Masukan kulkas…enak di santap ketika dingin
