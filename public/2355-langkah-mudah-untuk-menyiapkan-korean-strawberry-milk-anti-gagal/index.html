Cara membuat Korean strawberry milk:
  1. Potong kecil strawberry lalu masukan kedalam teflon, tambahkan gula pasir
  2. Nyalakan api kompor, lalu masak sambil diaduk aduk dan ditekan strawberry nya, masak sampai mengental, matikan api
  3. Penyajian, masukan selai tadi kedalam gelas beri dibagian pinggirnya juga
  4. Lalu tambahkan es batu dam tuangi susi cair, aduk rata sajikan
